O que eu sei fazer

Criação de Sites (Frontend)

  • HTML: Faço a estrutura de sites, como o esqueleto de uma página, para que tudo fique organizado e fácil de navegar.
  • CSS: Deixo os sites bonitos, com cores, fontes e layouts que funcionam bem em celulares e computadores.
  • JavaScript: Adiciono funções interativas, como botões que funcionam, formulários que respondem e conteúdos que mudam sem recarregar a página.
  • React: Crio sites modernos e rápidos, como aplicativos web, com partes que se atualizam sozinhas, como feeds de redes sociais.
  • Vue.js: Faço sites simples e rápidos de usar, com interfaces que reagem ao que o usuário faz.
  • Tailwind CSS: Uso para criar designs personalizados de forma rápida, mantendo tudo bem organizado e bonito.
  • Bootstrap: Utilizo para fazer sites com visual pronto e profissional, economizando tempo.

Sistemas e Bancos de Dados (Backend)

  • Node.js: Crio sistemas que fazem sites funcionarem, como salvar informações ou conectar com outros serviços.
  • Express: Uso para organizar o funcionamento de sites, como gerenciar logins ou enviar dados.
  • Python: Escrevo programas para automatizar tarefas ou criar sistemas, como ferramentas para organizar dados.
  • Django: Faço sites seguros com áreas de administração, como painéis para gerenciar conteúdo.
  • PostgreSQL: Organizo informações em bancos de dados, como listas de clientes ou produtos, de forma segura e rápida.
  • MongoDB: Gerencio dados de forma flexível, ideal para projetos grandes com muitas informações.
